写一下自己搭建博客使用的工具和方法，作为自己的第二篇博客。
这里使用的是hexo来搭建博客，利用插件实现博客的评论和计数功能。

hexo博客搭建笔记

依赖环境的安装

node环境的安装

一般推荐node使用官方网站的方法进行安装和更新，这里放一下node官网

我使用的是ubuntu，所以常见的更新方法就是在使用终端命令来安装npm专有的node版本管理模块。

npm instal -g n
n stable

当然这里有一个问题就是国内的nmp下载可能受到多重因素的影响，比较emmmm，所以这里推荐使用一个cnpm淘宝源。

npm install -g cnpm --registry=https://registry.npm.taobao.org

这里的cnpm以后就可以替代npm进行工作啦，用法和普通的npm语法一样例如cnpm info connect。

安装完毕的话，使用

node --version

来检验一下安装是否成功吧。

git环境的安装

一般我们使用apt命令即可安装git及其依赖

sudo apt install git-all

同样在安装之后检查一下。

git --version

hexo环境的安装

这里直接使用npm命令来安装hexo博客平台

npm install hexo-cli -g

或者按照上面安装了cnpm的同学也可以使用这个命令安装

cnpm install hexo-cli -g

主题安装

鉴于hexo的自带主题非常丑，所以这里也讲一下主题的安装。首先我们需要新建一个文件夹，这里以后就作为你的博客文件夹了，以后你的博客就是在这个目录里面建立起来的。进入文件夹，使用

hexo init

这个命令会在这里初始化博客文件夹，之后这里就可以clone其他的主题文件来修改博客主题了。这里推荐一个hexo主题网站，点开你喜欢的网站的github即可找到这个主题的克隆链接。

在命令行执行克隆链接，我这里以我网站使用的主题obsidian为例。

git clone https://github.com/TriDiamond/hexo-theme-obsidian.git theme/obsidian

然后执行

npm install

或者

cnpm install

补充一下可能遗漏的依赖包，这样主题就可以使用了。

修改配置文件

对博客页面配置文件的修改

这里修改的配置文件就是我们的博客主文件夹下的_config.yml文件。
在主文件夹打开终端，输入：

vim _config.yml

在vim打开配置文件，然后编辑其内容

# 省略上面的内容
theme: obsidian

然后使用hexo生成并且使用本地服务进行预览即可

hexo generate
hexo server

对主题配置文件的修改

打开obsidian文件夹下可以看到这里有另外一个_config.yml文件，这就是我们要修改的主题配置文件。

cd themes/obsidian
vim _config.yml

这里的修改可以参照同目录下的readme-cn文件

博客的推送

完成搭建后就可以通过github进行推送了。
